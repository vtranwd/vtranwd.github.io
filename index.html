My story

  • Spring 2016

    My first time coding

    I attended a full time Web Development Immersive boot camp at General Assembly and learned the tools and skills needed to become a junior full stack web developer after their 400 hour course. Pictured here is one of the first WordPress projects I created as a freelance developer.

  • 2016 - 2018

    My tech career takes off

    I've worked for a number of contractors and tech companies, including Alive Photo Studio as part of my WordPress experience.

  • 2018 - 2021

    Sudden career changes

    I made a dramatic shift in my career so that in 2018, I was cooking at various restaurants around Los Angeles, CA. Even without formal training, I rose the ranks and eventually started my own catering service, Smoky Pan Cooking.

  • 2021 - Present

    Renewed interest in EdTech

    During my time in the restaurant industry, I held onto my passion for languages and food insecurity, and I think that EdTech might be one way to address both of those issues.

  • Tools and Frameworks: JavaScript, jQuery, HTML, and CSS

SnapThis

Using MongoDB in a collaborative group project.

  • Project: Team project utilizing our newfound knowledge of MongoDB, Express, NodeJS, and Sockets.io
  • Tools and Frameworks: MongoDB, ExpressJS, NodeJS, Sockets.io, HTML, and Bootstrap CSS
  • Features: Users signed in with Google OAuth and submitted images via URL. Using sockets allowed real-time updates between users on different devices.
  • Story: As soon as I learned nodeJS and MongoDB at the General Assembly Web Development boot camp, I was assigned to a group where my knowledge of the tools was put to use to build this simple mobile-first app in a team environment.
